# 計算グラフ
テンソル（多次元配列）単位での演算をDAGで表現する。

## GraphOpについて
GraphOpは最適化の段階で最終的に融合されるので、最適化よりも演算子の種類を減らすことを重視する。そのため、Add, Negを組み合わせて減算を表現するようなことがある。

## View
Viewは、各軸の添え字からメモリオフセットに変換する写像を表現します。
この仕組みにより、ほぼゼロコストのView操作によって転置などが実現可能です。
